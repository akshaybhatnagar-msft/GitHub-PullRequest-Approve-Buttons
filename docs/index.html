GitHub PR Quick Actions

Add quick-action buttons to GitHub PRs for one-click reviews, approvals, and merges.

Add to Chrome Add to Edge View on GitHub

Features

Quick Reviews

Approve, comment, or request changes with a single click. Add optional comments via a clean modal.

Squash & Merge

Merge PRs instantly without navigating through dropdown menus.

Auto-Merge

Enable or disable auto-merge with one click. The button toggles based on current state.

Close PRs

Close pull requests directly from the header without hunting through menus.

Comment Count

See the number of review comments at a glance, right below the action buttons.

Dark Mode

Automatically matches GitHub's light or dark theme for a seamless experience.
