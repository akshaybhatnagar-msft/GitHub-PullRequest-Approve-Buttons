# Chrome Web Store / Edge Add-ons Listing

## Extension Name
GitHub PR Quick Actions

## Short Description (132 characters max)
Add quick-action buttons to GitHub PRs for one-click reviews, approvals, merges, and auto-merge - right in the PR header.

## Detailed Description

**Stop clicking through menus to manage pull requests.**

GitHub PR Quick Actions adds convenient buttons directly to your PR pages, letting you:

**Review PRs Faster**
• Approve with one click
• Leave comments quickly
• Request changes instantly
• All with an optional comment modal

**Merge & Manage**
• Squash and merge directly
• Enable or disable auto-merge
• Close PRs without hunting through menus

**Stay Informed**
• See comment counts at a glance
• Works with GitHub's dark mode

**How It Works**
1. Install the extension
2. Add your GitHub Personal Access Token (PAT)
3. Visit any GitHub pull request
4. Use the new Review and Actions buttons in the header

**Privacy First**
• Your token is stored securely in Chrome's extension storage
• No data is sent to third-party servers
• No analytics or tracking
• Open source: https://github.com/akshaybhatnagar-msft/GitHub-PullRequest-Approve-Buttons

**Required Permissions**
• Access to github.com and api.github.com to interact with PRs
• Storage to save your GitHub token locally

Perfect for developers, code reviewers, and teams who want to speed up their PR workflow.
